# Customer reply template

Per Playbook 06.

## Sizing

```
[Piece] fits:
Chest: [X] in
Length: [Y] in
Shoulder: [Z] in

Model wears [size].
```

No reassurance. Just numbers.

## Restock question

```
No restock.
```

That is the complete reply.

## Can you make this in [variation]?

```
No. This drop is closed.
```

## Order status / shipping

```
Your order ships by [date]. Tracking sent on dispatch.
```

## Complaint about tone

Do not reply.

Or, if a reply is warranted:

```
The position is intentional. It is not for everyone.
```

---

## Rules

- No apologies unless we are at fault.
- No "Hope this helps!" closers.
- No emoji.
