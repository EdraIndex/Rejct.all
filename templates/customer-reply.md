# Customer reply template

Per `PLAYBOOK_LIBRARY.md → Playbook 05` and Brand Bible §07, §11.

## Sizing

```
[Piece] — size [X]:
Chest: [X] in
Length: [Y] in
Shoulder: [Z] in

Model wears [size].
One size system, XS — XXL. Genderless.
```

No reassurance. Just numbers.

## Restock question

```
No restock.
```

Full reply.

## Can you make this in [variation / different size / different colour]?

```
No. This drop is closed.
```

## Payment — "Do you do COD?"

```
Prepaid only.
```

## Discount request

```
No discount codes.
```

## Shipping

```
Your order ships by [date]. Tracking sent on dispatch.
Free shipping above ₹3,999. Flat ₹149 below.
```

## Complaint about tone ("you're too cold", "no personality", etc.)

Do not reply.

Or, if a reply is warranted:

```
The position is intentional.
```

## Genuine return / defect

```
We'll make this right. Send two photos of the piece and your order number to [email].
Return label sent on confirmation.
```

---

## Rules

- No apologies unless we are at fault.
- No `Hope this helps!` or similar hospitality closers.
- No emoji.
- Brand name lowercase (`rejct.all`) in email.
- Hindi is optional in customer replies — use it if the customer wrote in Hindi or Hinglish, otherwise default English is fine for utility messages.
