# Press reply template

Per Playbook 03. Default: no.

## Default refusal

```
Thank you for thinking of us. We are not doing interviews at this time.
```

## Conditional yes (editorial only, written format, aligned outlet)

```
Thank you. We can answer in writing.

Questions in one document, replies in the same document, no live audio or video.
No photography requests beyond what we provide.
No product-placement arrangements.

If that works, send your questions.
```

## Advertorial / sponsored / celebrity-adjacent

```
Not a fit. Wishing you well.
```

---

## Rules

- Never open with "Hi [Name]".
- No exclamation points.
- Do not explain the position. They can find it on the site.
