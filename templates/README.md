# templates/

Reusable templates for recurring artifacts. Each template is a skeleton, not a draft.

## Files

- `product-copy.md` — product name + description
- `caption.md` — single-image caption
- `email.md` — infrequent, position-aligned emails
- `press-reply.md` — reply to press/interview requests
- `customer-reply.md` — reply to customer questions
- `drop-announcement.md` — waitlist-open and drop-live copy

## Rule

Templates are updated when the playbook they serve changes. Do not update a template without checking the relevant playbook first.
