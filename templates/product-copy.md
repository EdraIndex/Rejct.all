# Product copy template

Per Brand Bible §06 (material philosophy) and §09 (Drop 01 copy is the canonical example).

## Format

```
[SKU] — [Name · Colourway]
[Price]
[1–2 sentence description — what is on the garment, what carries the design. No story arc. No hype.]
[Material line — feeling, not specs.]
```

## Canonical examples (from Drop 01)

```
RJ-001 — Script Tee · Wine Red
₹2,800
Tonal wing embroidery at chest. rejct.all cursive script embroidered across the full upper back in wine red. The back is the statement.
Cotton. 220g. Heavy enough to mean something. Built to outlast the cycle.
```

```
RJ-006 — Angel Sleeveless · Tonal
₹3,000
Full tonal. Everything off-white on off-white. The angel seal is there. You have to look for it. The quietest and most considered piece in Drop 01.
Cotton. Off-white base. The graphic hides in plain sight.
```

## Rules

- Price in INR with the rupee symbol (`₹2,800`, not `Rs. 2800`).
- SKU format: `RJ-XXX`, sequential per drop.
- Name: 1–3 words + colourway after a middle dot. No version numbers.
- Description: what is literally on the garment, in plain language. Mention where (chest, back, collar, hem).
- Material line: follow Bible §06. Use feeling language. Never write `GSM`, `220g` is allowed where Bible uses it, never `premium`, `superior`, `luxury`.
- Hindi accompaniment: optional, at caption level (see `templates/caption.md`) — not typically in the product description itself.
