# Drop announcement template

Per Brand Bible §10 (Instagram launch phases) and `PLAYBOOK_LIBRARY.md → Playbook 01`.

Four phases: Before → Reveal → Drop → After. This template covers the core posts for each.

---

## Phase 1 — Before (2–3 weeks pre-drop)

One rejection per day, both languages. Example:

```
ट्रेंड को नकारो।
Reject the trend.
```

Plus standalone posts of the `R.` monogram. No product. No dates yet.

---

## Phase 2 — Reveal (1 week pre-drop)

Detail close-ups only. Caption is a single bilingual line or pure silence (image alone).

Example:

```
दिखावे को नकारो।
Reject performing.
```

---

## Phase 3 — Drop (drop day)

Full editorial shot per SKU, front and back.

Caption pattern:

```
[Drop name]. [Volume]. [Season].
[Line in Hindi, one line.]
[Line in English, one line.]
Link in bio.
```

Example:

```
Drop 01. Volume 00: Before. S/S 2026.
सब कुछ अस्वीकार करो।
Wear it like you mean it.
Link in bio.
```

No countdown, no hype.

---

## Phase 4 — After

Repost real people wearing it, no caption or a single bilingual line.

Once: `No restock. / कोई रीस्टॉक नहीं।`

Then begin Volume 01 teaser using the Phase 1 pattern for the next drop.

---

## Product page copy, per SKU

The Bible's Drop 01 entries (§09) are the canonical format:

```
[SKU] — [Name]
[Price]
[1–2 sentence description — what it is, what carries the design.]
[Material line — feeling, not specs.]
```

Example (RJ-005):

```
RJ-005 — Angel Sleeveless · Wine Red
₹2,800
Angel seal on back — cherub, halo, wings, puff R. at centre. Halo + wing at chest. REJECT on collar. The founding image of the brand, worn.
Cotton. Pure. The print does the rest.
```

---

## Never

- Countdown timers.
- "3 days to go / 2 days to go" sequences.
- BTS manufacturing reels framed as authenticity.
- Thank-you posts after sellout.
- Re-announce if the drop sold slowly.
- Hype words: `drop` (as hype), `collab`, `cop`, `fire`, `goes hard`.
