# Drop announcement template

Per Playbook 01.

## Waitlist-open (7 days before drop)

```
[Drop name]. [Date]. [Time] IST.
[One Hindi line or one English line.]
```

Example:

```
Refusal 01. 12 May. 03:11 IST.
यह पहनावा नहीं, रुख है।
```

## Drop-live (on site, not pushed out)

```
Live.
[Drop name].
No restock.
```

## 48-hours-after (one image post)

Caption: the drop's Hindi line only. Nothing else.

## Sold-through

Piece page stays up. Buy button replaced with:

```
बिक चुका / gone
```

---

## Rules

- No countdown timers.
- No "3 days to go" / "2 days to go" posts.
- No thank-you posts after sellout.
- No re-announce even if the drop sold slowly.
