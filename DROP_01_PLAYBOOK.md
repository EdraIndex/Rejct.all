# DROP 01 — Playbook

**Volume 00 · Before · S/S 2026.**
Governed by `Brand Bible.md` §08, §09, §10, §11.

This is a control document. Not a marketing brief. It defines what Drop 01 is, how it speaks, what it refuses, and what happens when it is questioned. Every Drop 01 decision routes through this file.

---

## 1. Drop Philosophy — why this drop exists

Drop 01 is the founding act. It is Volume 00: Before — the state before the rejection is spoken out loud. Six pieces. One purpose: install the design vocabulary the brand will operate under for every drop that follows.

Drop 01 is not a launch. It is the position, materialised in cotton. Every piece carries the `R.` monogram — the constant. Every piece answers to the Six Rejections.

**What Drop 01 is for:**
- Installing the visual codes — off-white base, wine red embroidery, angel seal, script back, `REJECT` collar, puff `R.`, tonal expression.
- Marking the first aligned buyers.
- Proving the brand keeps its refusals. No restock. No discount. No hype. No COD.

**What Drop 01 is not for:**
- Proving the brand. The Bible proves the brand.
- Building a customer base. It selects one.
- Experimenting. The design language is established, not explored.

## 2. Product Naming — no fashion-industry patterns

**Allowed format — locked:**
`[SKU] — [Name] · [Colourway]`

**Drop 01 names and SKUs — locked, per Bible §09:**

| SKU | Name |
| - | - |
| RJ-001 | Script Tee · Wine Red |
| RJ-002 | Script Tee · Olive |
| RJ-003 | Loop Tee · Wine Red |
| RJ-004 | Circle Sleeveless · Wine Red |
| RJ-005 | Angel Sleeveless · Wine Red |
| RJ-006 | Angel Sleeveless · Tonal |

**Banned naming patterns:**
- Season codes in the product name (`SS26-...` allowed in SKUs only).
- Fashion-house borrowings — `The Parisian`, `Le Monde Tee`, `Roma Crew`.
- Narrative names — `Becoming`, `Solitude`, `The First Step`, `Manifesto Tee`.
- Adjective names — `Premium Tee`, `Essential Tee`, `Signature Tee`, `Statement Tee`.
- Gendered names — `The Boyfriend Tee`, `The Girlfriend Tee`.
- Region-borrowed names — `Tokyo Tee`, `Delhi Edit`, `Bombay Heavy`.
- `Unisex` prefix. Banned outright — the brand is genderless, not unisex.
- Version suffixes inside a drop (`.v1`, `.01 Remastered`).
- Founder / collaborator names.

**The rule:** the name describes the construction (Script, Loop, Circle, Angel) or the silhouette (Tee, Sleeveless). Colourway is the second identifier. That is the whole naming system.

**RJ-004 — Circle Sleeveless · Wine Red — ₹2,600**
`REJECT` on collar. Puff `R.` at chest. Circular `rejct.all` ring + oversized puff `R.` on back. The most graphic sleeveless.

**RJ-005 — Angel Sleeveless · Wine Red — ₹2,800**
Angel seal on back — cherub, halo, wings, puff `R.` at centre. Halo + wing at chest. `REJECT` on collar. The founding image of the brand, worn.

**RJ-006 — Angel Sleeveless · Tonal — ₹3,000**
Full tonal. Everything off-white on off-white. The angel seal is there. You have to look for it. The quietest and most considered piece in Drop 01.

**Banned in every product description:**
- Adjective stacks — `premium`, `luxurious`, `superior`, `high-quality`.
- Fabric specs — `GSM`, blend percentages.
- Style claims — `timeless`, `essential`, `versatile`.
- Emotional sell — `feel your best`, `look incredible`.
- Care instructions — those live on a tag, not in the description.
- Fit commentary beyond what is physical fact — "drapes beautifully," "flatters every figure" are rejected.

Prices are stated. They are not justified.

| SKU | Piece | Price |
| - | - | - |
| RJ-001 | Script Tee · Wine Red | ₹2,800 |
| RJ-002 | Script Tee · Olive | ₹2,800 |
| RJ-003 | Loop Tee · Wine Red | ₹3,200 |
| RJ-004 | Circle Sleeveless · Wine Red | ₹2,600 |
| RJ-005 | Angel Sleeveless · Wine Red | ₹2,800 |
| RJ-006 | Angel Sleeveless · Tonal | ₹3,000 |

**Commerce rules — locked per Bible §11:**
- No sale pricing. Ever.
- No discount codes. Ever.
- No promotional pricing. Ever.
- Tonal colourway ≥ equivalent graphic colourway. (RJ-006 > RJ-005 — correct.)
- Free shipping above ₹3,999. Flat ₹149 shipping below.
- No COD. Prepaid only.

**The brand does not:**
- Explain the price.
- Justify the price with fabric cost, labour cost, or design cost.
- Benchmark against competitors.
- Call the price `accessible`, `affordable`, `premium`, `luxury`, or `investment`.
- Apologise for the price.
- Discount for friends, press, influencers, collaborators, employees, family. No one.

**Authorised reply to "why this price":**
```
This is the price.
```

No other reply is authorised.

## 5. Drop Messaging — Before / During / After

Per Bible §10 four-phase model. The phases are non-negotiable.

### 5.1 Before (2–3 weeks pre-drop)

**Objective:** build the idea before the object.

**Allowed:**
- One rejection per day, bilingual, each line ≤ 8 words.
- `R.` monogram as a standalone post.
- Text-only posts.
- Silence days — posting nothing for a day is a valid move.

**Forbidden:**
- Any product image or fragment.
- Any date tease.
- Any `coming soon` graphic beyond the `R.`
- Any BTS manufacturing content.
- Any countdown graphic or story.

**Caption sample:**
```
ट्रेंड को नकारो।
Reject the trend.
```

### 5.2 Reveal (1 week pre-drop)

**Objective:** reveal only what the wearer will see up close.

**Allowed:**
- Thread-texture close-ups.
- `REJECT` collar macro shots.
- Puff `R.` dimensional detail.
- Script curve close-ups.
- Ghost-mannequin animations — detail only, never full silhouette.

**Forbidden:**
- Full-garment photography.
- Model-worn photography.
- Styling shots.
- Reels with audio beyond ambient.

### 5.3 During — Drop Day

**Objective:** publish. Do not announce.

**Post structure, per SKU:**
- One editorial image, front.
- One editorial image, back.
- One single caption, bilingual, locked structure:

```
Drop 01. Volume 00: Before. S/S 2026.
सब कुछ अस्वीकार करो।
Wear it like you mean it.
Link in bio.
```

**Forbidden:**
- Countdown posts.
- Hype sequences.
- Repeated `it's here` across the day.
- Press-drama embargo lifts.
- Story highlights titled `NEW`, `DROP`, `BUY NOW`, `LIVE`.

### 5.4 After

**Objective:** absorb the wearing. Do not rally.

**Allowed:**
- Reposts of real buyers wearing the pieces — no caption, or one bilingual line.
- One reminder post: `No restock. कोई रीस्टॉक नहीं।`
- Begin Phase 1 (Before) for Volume 01.

**Forbidden:**
- Thank-you posts.
- `So grateful` language.
- `Sold out in X minutes` posts.
- Re-announcements if the drop sold slowly.
- Any suggestion a restock might happen.

**Sold-through marker on each product page:**
```
बिक चुका / gone
```

Replaces the buy button. Permanent.

## 6. What We Will Not Do — Drop 01 exclusion list

None of the following is an option. Not at any pressure level. Not "just this once." Not for anyone.

- **No countdowns.** No `3 days to go`, no `24 hours`, no timers on the site, no dates flashed across stories.
- **No "limited stock" framing.** The pieces are final, not rare. `No restock.` is the only sentence allowed in this territory. Forbidden: `limited edition`, `only X available`, `rare`, `exclusive`, `collector's`, `just X pieces made`.
- **No influencer push.** No paid creators. No gifting for reach. No seeding selected by follower count.
- **No urgency triggers.** No `last chance`, no `ends today`, no `hurry`, no cart timers, no `someone just bought this` tickers, no stock-level counters, no abandoned-cart guilt emails.
- **No discount mechanics.** No codes. No first-order offer. No referral bonus. No bundle pricing. No student discount.
- **No celebrity seeding.** Even if free. Even if they ask.
- **No marketplace listings.** Drop 01 is on `rejctall.com`. That is the only surface.
- **No retail partners.** Online-only at launch.
- **No press-embargo drama.** Editorial placements, where they happen, are quiet.
- **No sales moments.** Not for Diwali, festive, end-of-season, Black Friday, or any calendar holiday.
- **No founder-led campaign content.** No letter from the founder. No video explaining the vision. The brand is the position, not the person.
- **No lifestyle content.** No morning routines. No `my week in Drop 01`. No aesthetic boards.
- **No hashtag campaigns.** No branded hashtag. No UGC prompt.
- **No `selling fast` signal.** Not in copy, not in visuals, not in implication.

This list is closed. Additions require a decision record in `docs/decisions/`.

## 7. Customer Reaction Expectation

**Confusion is acceptable. Clarity is not required.**

### Expected reactions from aligned buyers
- Recognition — *"this is me."*
- Silent purchase.
- Wearing the piece and telling no one.
- Quiet word-of-mouth between aligned people.

### Expected reactions from unaligned onlookers
- Confusion — *"what is this brand?"*
- Dismissal — *"this isn't for me."*
- Mild offence — *"why so cold?"*
- Irritation at the refusal to explain.

**None of the above is a problem.**

### The brand will not
- Explain further in response to confusion.
- Warm the tone to soften dismissal.
- Apologise for the coldness.
- Pivot register toward mass appeal.

### Authorised replies to common unaligned reactions

| Reaction | Reply |
| - | - |
| *"You're so cold."* | `The position is intentional.` |
| *"I don't get it."* | `It isn't for you.` |
| *"Why don't you explain the brand?"* | `We don't.` |
| *"You're going to lose customers."* | No reply. Not every DM requires one. |
| *"What does the angel mean?"* | `The image carries itself.` |
| *"Why is the price this high?"* | `This is the price.` |

**Operating principle:** a confused onlooker is a feature, not a bug. The confusion filters the audience into aligned and unaligned. Drop 01 is the filter, not the fix.

## 8. Enforcement

Every Drop 01 decision routes through, in order:

1. **`Brand Bible.md`** — source of truth.
2. **This playbook** — Drop 01 specifics.
3. **`CLAUDE.md`** — voice and output gate.
4. **`PLAYBOOK_LIBRARY.md → Playbook 01`** — four-phase launch procedure.
5. **`BRAND_SYSTEM_MAP.md`** — the Six Rejections matrix for edge cases.

**Conflict resolution:**
- This document vs. the Bible → Bible wins.
- This document vs. a user request → this document wins, unless the user cites a specific Bible section that overrides it.

**Not overrides:**
- "I changed my mind."
- "We need revenue."
- "This one time."
- "Everyone does it."
- "It will grow the audience."

These are rejected on sight. The position is not a negotiation.
