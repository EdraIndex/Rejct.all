# output/

Generated artifacts from prompts and templates live here.

## Structure

Organize by type or by drop — pick one and stay consistent.

Suggested:

```
output/
  drops/
    [drop-name]/
      copy.md
      captions.md
      emails.md
  copy-reviews/
    YYYY-MM-DD-[subject].md
  refusals/
    YYYY-MM-DD-[request-type].md
```

## Rule

Output is disposable. If an artifact becomes load-bearing, promote it to `docs/` or `templates/`.
