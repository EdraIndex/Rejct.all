# prompts/

Reusable prompts for Claude. Each prompt is a file. Paste or reference it in a conversation.

## Naming

`NN-short-title.md` — numbered so related prompts stay together.

## Structure

Each prompt file contains:

1. **Purpose** — one line, what this prompt produces.
2. **Input** — what to give Claude before running it.
3. **Prompt** — the prompt itself, copy-paste ready.
4. **Output expectations** — what a good response looks like.

## Current prompts

- `01-tone-check.md` — run any draft copy through the tone rules
- `02-draft-product-copy.md` — generate product name + description
- `03-draft-hindi-line.md` — generate a standalone Hindi line
- `04-reject-request.md` — draft a terse refusal
- `05-drop-plan.md` — generate a drop launch plan per Playbook 01
