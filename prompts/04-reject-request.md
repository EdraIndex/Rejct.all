# Reject request

## Purpose
Draft a terse refusal to a press, collab, partnership, or customer request.

## Input
- What kind of request (press, collab, customer, other)
- Verbatim request if available
- Tone of reply (neutral, warmer, firm)

## Prompt

```
Draft a refusal reply for Rejct.all. Follow CLAUDE.md tone rules and Playbook 07 (Saying no).

Request type: [press / collab / customer / other]
Request: [paste request or summarize]
Tone: [neutral / warmer / firm]

Constraints:
- Maximum 3 sentences.
- No apology unless the request was in good faith and aligned.
- No explanation of the brand position unless explicitly asked.
- No softening adverbs ("unfortunately", "sadly").
```

## Output expectations
- A reply that could be sent as-is.
- Shorter than the request.
- Not defensive. Not explanatory.
