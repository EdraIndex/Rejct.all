# second-brain/

References, research, saved inputs, and unshipped ideas.

Anything that informs the work but is not the work itself lives here.

## Structure

- `references/` — things we read, watch, or save. Each file is one source with a note on why it matters.
- `research/` — open threads. Market observations, audience signals, cultural currents.
- `ideas/` — things we might do. Ideas die here quietly if they do not survive the OS.

## Rules

1. Every file has a one-line purpose at the top.
2. If something here sits untouched for 90 days, delete it. It was not load-bearing.
3. This is not an archive. It is a holding pattern.
